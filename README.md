# Corporate Tax Depreciation Engine

A Python-based interactive web application built with Streamlit that calculates and visualizes corporate asset depreciation schedules. This tool was developed to bridge financial accounting principles with business analytics and data visualization.

## Features
* **Dynamic Calculations**: Instantly compute both **Straight-Line** and **Double Declining Balance** depreciation schedules.
* **Interactive UI**: Easily adjust initial asset cost, salvage value, and useful life through a clean sidebar interface.
* **Financial Data Tables**: Generates a comprehensive, year-by-year depreciation schedule detailing expense, accumulated depreciation, and ending book value.
* **Visual Trajectories**: Includes automated Matplotlib visualizations tracking the asset's book value and depreciation expense over its lifecycle.

## Technology Stack
* **Python**: Core logic and calculations
* **Streamlit**: Frontend web application framework
* **Pandas**: Data manipulation and schedule structuring
* **Matplotlib**: Financial charting and visualization

## Installation & Usage

To run this engine locally on your machine:

1. **Clone the repository**
   ```bash
   git clone https://github.com/santos-150/tax-depreciation-engine.git
   cd tax-depreciation-engine

2. **Install dependencies**
   Ensure you have Python installed, then run:
   ```bash
   pip install streamlit pandas matplotlib
   ```

3. **Launch the application**
   ```bash
   streamlit run app.py
   ```
